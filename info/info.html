About FFTabClose

Features

FFTabClose helps you keep your browser organized by automatically closing tabs you haven't used in a while, across all browser windows and workspaces.

How the Timer Works

Each tab has its own timer that starts when it's opened or activated. Tabs are closed or discarded automatically once they exceed your specified time limit. Your settings persist even when restarting the browser.

Tab Exclusions

Tabs that are playing audio are never closed or discarded. Pinned tabs are not closed but can be unloaded to save memory. Currently active tabs in any window are never closed.

Understanding Tab Discarding

Discarding frees up memory while keeping the tab visible. Tabs automatically reload when you click on them.

Privacy & Permissions

FFTabClose requires the following permissions:

  • tabs To monitor and manage tabs across all windows
  • storage To save your preferences and tab timers
  • alarms To schedule periodic tab checks
  • contextualIdentities To support Firefox containers and workspaces
  • cookies Required for container integration

Domain Rules

You can set custom rules for specific domains to:

  • Never close tabs from certain sites
  • Always close tabs from specific domains
  • Set custom timeouts for particular websites

Configure domain rules in the extension popup by clicking the "+" button.
